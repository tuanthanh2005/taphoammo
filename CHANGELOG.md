All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-30

### Added
- ✅ Hệ thống đăng nhập/đăng ký
- ✅ Đăng nhập bằng Google OAuth
- ✅ Phân quyền: Admin, Seller, User, Affiliate
- ✅ Quản lý sản phẩm số (key, account, file, link)
- ✅ Hệ thống giỏ hàng và thanh toán
- ✅ Ví nội bộ
- ✅ Tự động giao hàng sau thanh toán
- ✅ Hệ thống rút tiền cho Seller
- ✅ Phí admin 5% trên mỗi đơn hàng
- ✅ Phí rút tiền 5%
- ✅ Dashboard cho Admin, Seller, User
- ✅ Quản lý kho hàng số
- ✅ Đánh giá sản phẩm
- ✅ Hệ thống affiliate
- ✅ Lịch sử giao dịch
- ✅ CSRF protection
- ✅ SQL Injection protection
- ✅ XSS protection
- ✅ Rate limiting login
- ✅ Responsive design với Bootstrap 5

### Security
- Password hashing với bcrypt
- PDO Prepared Statements
- CSRF token validation
- Session security
- File upload validation

### Performance
- Database indexing
- Pagination
- Optimized queries
- LiteSpeed Cache compatible
